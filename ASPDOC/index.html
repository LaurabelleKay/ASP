Block Timetabler(Check the source code of block Timetabler)

Timetabler which creates a tmetable based on an input file

@block DEFAULT
  |_@block Timetabler
  |_@block Schedule
  |_@block Student
  |_@block Lecturer
  |_@block Room
  |_@block Unit

Input Atoms

  • student(S)
    A student
    • S: A student number
  • unit(U)
    A unit
    • U: A unit code/name
  • lecturer(L)
    A lecturer
    • L: The name of a lecturer
  • room(R)
    A room
    • R: A room number
  • takes(S, U)
    Student S takes unit U
    • S: A student number
    • U: A unit code/name
  • teaches(L, U)
    Lecturer L teaches unit U
    • L: The name of a lecturer
    • U: A unit code/name
  • capacity(R, Z)
    Room R has capacity Z
    • R: A room number
    • Z: The number of students a room can hold
  • preferred(L, T, D)
    Lecturer L's preferred teaching times
    • L: The name of a lecturer
    • T: A time in the day which ranges from 09:00 to 18:00
    • D: A day in the week ranging form Monday to Friday
  • day(Gringo Interval Term(1,5))
    number of teaching days
    • timestep(Gringo Interval Term(9,18))
      range of teaching hours defined as 9am to 6pm

      Output Atoms

      • lec(U, R, L, T, D)
        Scheduled lecture for a unit
        • U: A unit code/name
        • R: A room number
        • L: The name of a lecturer
        • T: A time in the day which ranges from 09:00 to 18:00
        • D: A day in the week ranging form Monday to Friday

  |_@block Student
  |_@block Lecturer
  |_@block Room
  |_@block Unit

      Preconditions

      PreferredClash: A lecturer cannot have preferred lecturetime that is the same asan unavailable time

      Postconditions

      LecturerClash: A lecturer cannot teach 2 lectures at the same time
Unavailable: A lecturer cannot lecture when they are unavailable
wrongUnit: A lecturer cannot have a lecture for a unit they do not teach
